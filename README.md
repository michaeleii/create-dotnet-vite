# create-dotnet-vite

A CLI tool for quickly scaffolding a new .NET 8 web application with Vite as the front-end build tool.

## Usage

1. Run it with `bunx`.

```bash
bunx create-dotnet-vite
```

## Run it locally

1. Install the dependencies.

```bash
bun i
```

2. Run the following command to run the CLI tool.

```bash
bun start
```
